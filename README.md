# AI Business Analyst

An AI-powered Business Intelligence platform that transforms Excel and CSV sales data into actionable business insights.

## Project Status

🚧 MVP in Planning Phase

## MVP Features

- Upload Excel and CSV files
- Data cleaning and validation
- Sales KPI generation
- Top-selling products analysis
- Sales by region
- Monthly sales trends
- Slow-moving products detection
- Customer performance analysis
- Interactive dashboards
- Executive summary generation

## Tech Stack

### Frontend
- React
- JavaScript

### Backend
- Python
- FastAPI

### Data Processing
- Pandas
- DuckDB
- SQL

### Visualization
- Recharts

## Project Goals

- Transform raw sales data into business insights.
- Reduce manual Excel analysis.
- Help managers make faster business decisions.
